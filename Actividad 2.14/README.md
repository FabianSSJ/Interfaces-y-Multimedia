# Proyecto Eco-Interactivo XR | Actividad 2.14

## Descripción del Proyecto

Este proyecto es un entorno XR interactivo desarrollado con A-Frame para la asignatura de Interfaces y Multimedia. El objetivo es concienciar sobre el impacto ambiental mediante una narrativa dinámica y un sistema de accesibilidad universal robusto.

## Hitos Logrados

- **Narrativa por Presencia**: Uso de Raycasting para detectar la mirada del usuario durante 10 segundos, activando cambios en la iluminación y paneles informativos.
- **Accesibilidad WCAG**: Menú de preferencias para Alto Contraste (1.4.3), Subtítulos Espaciales (1.2.2) y Foco Visible (2.4.7).
- **Optimización**: Escena fluida a 60 FPS mediante el uso de primitivas geométricas y texturas comprimidas.
